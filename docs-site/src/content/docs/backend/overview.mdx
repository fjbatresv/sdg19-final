## Endpoints

- `POST /auth/register`
- `POST /auth/login`
- `POST /auth/refresh`
- `GET /products` (cacheable en CloudFront)
- `POST /orders`
- `GET /orders`

## Flujo de ordenes

```mermaid
sequenceDiagram
  participant API as API Gateway
  participant Orders as Orders Lambda
  participant DDB as DynamoDB
  participant Stream as DynamoDB Stream
  participant OrderStream as Order Stream Lambda
  participant SNS as SNS Topic

  API->>Orders: Create order
  Orders->>DDB: PutItem
  DDB->>Stream: Stream record
  Stream->>OrderStream: Invoke
  OrderStream->>SNS: Publish
```
