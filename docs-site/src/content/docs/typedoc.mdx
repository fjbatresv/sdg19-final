---
title: TypeDoc (Backend)
---

TypeDoc genera la documentación de la API del backend.

## Generar documentación local

```bash
npx nx run docs:api
```

Salida: `docs/api-typedoc/`.

## Validar en CI

```bash
npx nx run docs:check
```

El target `docs:check` ejecuta la cobertura de JSDoc y TypeDoc.
