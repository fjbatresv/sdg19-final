Este proyecto incluye análisis de costos con y sin VPC:

- `VPC-COST.md`: costos con NAT gateways, tráfico y servicios dentro de VPC.
- `NO-VPC-COST.md`: costos sin VPC, reduciendo gastos fijos de red.

Resumen rápido:

- Con VPC: se consideran 3 NAT gateways, rutas y tráfico saliente; el costo base
  es mayor, pero habilita acceso a subredes privadas y controles de red más estrictos.
- Sin VPC: elimina NAT gateways y reduce el costo mensual; recomendable para
  ambientes de desarrollo o cargas bajas sin requerimientos de red privados.
