Este proyecto incluye análisis de costos con y sin VPC:

- `VPC-COST.md`: costos con NAT gateways.
- `NO-VPC-COST.md`: costos sin VPC.

Recomendación:

- Para desarrollo, desactivar VPC reduce el costo base.
- En producción, evaluar el costo de NAT vs. requerimientos de red.
