Este proyecto incluye analisis de costos con y sin VPC:

- `VPC-COST.md`: costos con NAT gateways.
- `NO-VPC-COST.MD`: costos sin VPC.

Recomendacion:

- Para desarrollo, desactivar VPC reduce el costo base.
- En produccion, evaluar el costo de NAT vs. requerimientos de red.
